    Histroy

    Version 0.1

          其实最早有在网络上建个站子的想法早在初二时。那时候周末常常去网吧上网浏览一些网站。时2001年，个人建站正是最风行的时候，网易、搜狐、新浪等门户都有专门的板块介绍建站以及提供免费的空间。其实无非是可以往一些模板里填一些字罢了，但在当时，自己排列的几个汉字以宋体呈现在浏览器上，还能以网址访问，足以让初二的小屁孩激动了。到高一时制作了一个简单的site，名为“半成年”，挂在网易的免费空间下。由于网易的免费空间服务中止的缘故，只存活了短短一个月。
          MileStone:首次上线

    Version 0.5

          之后，Web2.0席卷互联网。在大学前那个暑假时也加入了草根博客大军。在新浪申请了个博客，记录些随笔随想。那时候很喜欢一段词：“柴门半掩清寒逸，寂寞苔檐碧。东风也欲报君知，又是莺歌蝶舞弄春时。”故名为“苔檐之居”。进入后台，最自由的布局只是在侧边栏加减一个模块，最个性的域名不过是尾缀一个用户名。厌倦之心渐起。再者，前面被网易突然中止服务的经历也让我对网络的免费服务有点阴影。偶然间看到朋友的独博客后台，心生向往：丰富的管理功能、自由的页面设计、个性的域名。
          MileStone:檐  版的首出

    Version Alpha

          大一时还不能带电脑，是在学校的图书馆电子阅览室，开始选择博客程序，挑空间服务商。由于名字上的亲切，买了空间后在上面部了一个基于php的Bo-blog。由于对php的陌生，期间的安装和绑定什么的都是现学现卖。当我辛苦地从新浪搬完文后，又见异思迁起来：那段时间开始学习asp，看着尖括号加%要比尖括号加？的语法要亲切地多。开始埋怨自己选了个不是asp的博客。于是懒得扔在一边。也因为学业上忙碌的原因，继续在新浪上放文。――这算是檐在内测阶段流产掉的一个版本。
          MileStone:独立的主机和域名

    Version Beta

          往后，便到了大二的暑假。重新买空间选程序，这次是用z-blog。那个星期，忙得不亦乐乎。一周后出了效果，美工上还是差强人意。不由感叹毕竟不是科班出身。而且关键是，以那个风格根本就不可能衍化至整个站点。起码我在做摄影展示页时就选择了截然相反的风格――黑色调更容易处理。做完这个皮肤和摄影页，挂在了现在这个域名下。名为苔檐。
          MileStone:启用yecols.cn域名

    Version 1.0

          再到后来，越来越觉得要有一个自己站子的必要性，非但是由于想挂载一个博客的原因，还因为想将做过的东西做个小小的展示，标记下我曾做过的努力。而不是让它们沉睡在电脑硬盘的角落。另外，往往看到很多希望有所涉猎的技术，比如Silverlight、air、SEO，以及一些流行的js库，很想实践，却苦于没有它们的用武之地。以上，便是本站 portfolio 和 lab 模块出现的缘由。这一次，缘于自己还算满意的页面设计，得以坚持到底。给自己定期限，埋头做了多日。终于上线。是为现在的“檐”。
          MileStone:1.0版本上线
